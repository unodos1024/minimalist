# 심플하게 산다 (L'Art de la Simplicité)

> "단순함은 궁극의 정교함이다." — 레오나르도 다 빈치

**심플하게 산다**는 도미니크 로로의 철학에서 영감을 받아 제작된 미니멀리즘 라이프스타일 앱입니다. 복잡한 세상 속에서 불필요한 것을 덜어내고, 본질에 집중할 수 있도록 돕습니다.

## ✨ 주요 기능

### 1. 성찰의 시간 (Daily Reflection)
매일 새로운 질문을 통해 내면을 들여다봅니다. 복잡한 마음을 문장으로 정리하며 해소합니다.

### 2. 공간 비우기 (Declutter Timer)
타이머와 함께 물리적 공간을 정리하는 시간을 갖습니다.
- **앰비언트 사운드**: 바람 소리, 빗소리 등 자연의 소리를 들으며 비움에 집중할 수 있습니다. (모바일 최적화 완료)
- **비움 기록**: 비워낸 물건을 기록하면, 그만큼의 '무게'가 삶에서 덜어졌음을 시각화해 줍니다.

### 3. 의도적 멈춤 (Mindful Pause)
충동적인 소비 욕구가 들 때, 24시간의 유예 시간을 가집니다. 욕망이 정말 필요한 것인지, 아니면 일시적인 기분인지 스스로 질문하게 합니다.

### 4. 기록 보관소 (Archives)
비워낸 물건과 생각들을 보관합니다. 비워낸 것들이 쌓일수록 앱의 배경은 점점 더 밝고 가벼워집니다.

### 5. 다국어 및 테마 지원
- **언어**: 한국어, English, Français, Español, 日本語
- **테마**: 눈이 편안한 '정오 모드(Light)'와 깊은 몰입을 위한 '자정 모드(Dark)'

---

## 🛠 기술 스택

이 프로젝트는 **극도의 미니멀리즘**을 코드에도 적용하여, 외부 라이브러리 없이 순수 웹 기술만으로 구현되었습니다.

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (No Framework)
  - Single File Component (`index.html`) 구조
  - `localStorage`를 이용한 데이터 영구 저장
  - Web Audio API를 활용한 프로시저럴 앰비언트 사운드 생성

## 📂 프로젝트 구조

```
index.html       # 웹 버전 소스 (핵심 로직 포함)
README.md
```

`web/index.html` 파일을 크롬 등 웹 브라우저에서 직접 열면 바로 실행됩니다.
